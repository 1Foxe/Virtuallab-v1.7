bagaimana Cara Merubah sandi Vlab Polibatam?

Ada 2 jenis perubahan kata sandi, sebagai berikut:

    1. Change Password

      Anda dapat merubah password pada halaman dashboard menuju icon user pada pojok kanan atas, kemudian pilih menu change password.

      lalu masukkan password lama dan password baru anda dan klik button simpan.


    2. Forgot password

      Jika anda lupa sandi, Anda dapat merubah password melalui fitur forgot password pada halaman login.

      Selanjutnya anda akan diminta untuk memasukkan email yang anda gunakan pada saat anda melakukan registrasi akun, untuk mendapatkan kode verifikasi.

      masukkan kode verifikasi yang telah dikirim melalui email anda.

      Setelah melakukan verifikasi, maka anda akan di minta untuk memasukkan kata sandi yang baru.
