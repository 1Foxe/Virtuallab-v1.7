Bagaimana Cara Install Vlab Polibatam?

VLAB Polibatam menyediakan install aplikasi agar mudah di gunakan tanpa perlu lagi mengakses web , dengan menggunakan fitur PWA (Progressive Web Apps) . VLAB Polibatam dapat di install di berbagai Device , antara lain :

  1. Desktop

    2. pada bagian kanan chrome terdapat opsi untuk menginstall aplikasi ke dekstop pada PC atau laptop anda, lalu klik install untuk menambahkan aplikasi ke dalam dekstop


    Jika sudah Terinstall untuk tampilan dekstop maka akan seperti ini


    Berikut hasil tampilan icon pada dekstop

  2. Smartphone

    3. pada bagian bawah chrome, klik notifikasi untuk menginstall aplikasi ke halaman utama pada smartphone anda.

    Jika sudah Terinstall untuk tampilan smarthphone maka akan seperti ini.

    Berikut hasil tampilan icon pada smarthphone.
